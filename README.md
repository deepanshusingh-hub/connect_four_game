# Instagram Clone

A clone of instagram with implemented functionalities of uploading posts, comment , authentication.
Technology used - firebase, react.js

#### Clone the project

```bash
  git clone https://link-to-project
```
#### Running server
````
$ cd my-project
$ npm start

````
